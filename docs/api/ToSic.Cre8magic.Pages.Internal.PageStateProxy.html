Wrapper for PageState, so that we can "inject" a debug state to a specific menu. This is just for internal development.

Goal is that if we have a page with a lot of test-menus, we can provide this instead of the real PageState, and then we can stop in certain places - only for this menu.

To do that, see the extension methods below.

Usage ca.: if (PageState.IsDebug()) PageState.DoNothing(); // set breakpoint here
